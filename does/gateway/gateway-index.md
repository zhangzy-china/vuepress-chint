####  **网关状态**

|   类别状态  |   说明  |
| --- | --- |
|   在线  |    网关正常运行   |
|    离线 |   网关通讯中断  |
|    异常 |    网关非正常工作状态 |

####  **图表概览**

* 网关类型：统计账户下网关类型数量。
* 网关状态：参照网关状态。
* 固件版本：网卡软件版本统计。
